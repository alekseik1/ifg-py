Summary
=======
Numerical calculation of ideal Fermi gas (IFG) properties.

Currently supports temperatures in range [1e-49..1e+49] and
specific volumes in range [1e-30..1e+20].
Others are not tested.


Installation
============
Easily install via ``pip``::

    pip install ifg-py
